# Backend Developer - HushRyd Platform

## Company Overview
HushRyd is an innovative ride-sharing platform connecting drivers and passengers for safe, reliable intercity travel. We're building technology that makes transportation accessible, affordable, and secure across India.

## Position: Backend Developer
**Experience:** 1.5 - 2 Years  
**Employment Type:** Full-Time  
**Location:** [Specify Location] / Remote  

---

## About the Role
We're looking for a passionate Backend Developer to join our engineering team. You'll work on building and maintaining scalable APIs, real-time features, and core platform services that power our web and mobile applications. This is an excellent opportunity for someone who wants to grow their skills while working on a product that impacts thousands of users daily.

---

## Key Responsibilities

### API Development & Maintenance
- Design, develop, and maintain RESTful APIs using Node.js and Express.js
- Implement authentication and authorization systems (JWT, OTP-based login)
- Build and optimize database queries with MongoDB/Mongoose
- Develop real-time features using Socket.IO for live tracking and notifications

### Feature Development
- Implement booking, search, and trip management functionalities
- Build driver registration and document verification workflows
- Develop payment integration and fare calculation services
- Create notification systems (SMS, Email, Push, WhatsApp)

### Code Quality & Testing
- Write clean, maintainable, and well-documented code
- Implement unit tests and property-based tests using Jest
- Participate in code reviews and follow best practices
- Debug and resolve production issues efficiently

### Collaboration
- Work closely with frontend and mobile app developers
- Collaborate with product team to understand requirements
- Participate in sprint planning and daily standups
- Document APIs and technical specifications

---

## Required Skills & Qualifications

### Technical Skills
- **Node.js & Express.js:** Strong proficiency in building REST APIs
- **MongoDB:** Experience with Mongoose ODM, aggregation pipelines, indexing
- **JavaScript (ES6+):** Solid understanding of async/await, promises, closures
- **Git:** Version control with branching strategies (Git Flow)
- **REST API Design:** Understanding of RESTful principles and best practices

### Good to Have
- Experience with Socket.IO or WebSockets for real-time applications
- Familiarity with AWS services (S3, EC2, ECS)
- Knowledge of Docker and containerization
- Experience with CI/CD pipelines (GitHub Actions)
- Understanding of security best practices (Helmet, CORS, encryption)
- Exposure to payment gateway integrations

### Soft Skills
- Strong problem-solving and analytical abilities
- Good communication skills (written and verbal)
- Ability to work independently and in a team
- Eagerness to learn new technologies
- Attention to detail and code quality
